# 🖥️ Exercícios de Front-End Essencial — Serratec

Repositório com os exercícios práticos desenvolvidos durante o módulo de **Front-End Essencial** do programa de formação do [Serratec](https://serratec.org/).

Cada exercício explora conceitos fundamentais de HTML e CSS, com foco em estruturação de páginas, estilização e responsividade.

## 📝 Exercícios

### Exercício 01 — Página de Perfil Pessoal

Uma página de apresentação pessoal com foto de perfil, texto de introdução, lista de interesses e links para redes sociais.

**Conceitos praticados:**
- Estruturação semântica com HTML
- Flexbox para centralização e layout em coluna
- Estilização de imagem circular com `border-radius`
- Links estilizados como botões
- Paleta de cores personalizada e `text-shadow`

---

### Exercício 02 — Loja Virtual

Uma vitrine de produtos de informática (monitor, teclado e mouse) com nome, descrição, preço e botão de compra.

**Conceitos praticados:**
- Layout em linha com Flexbox (`flex-direction: row`)
- Cards de produto estilizados
- Responsividade com `@media query` (quebra de coluna abaixo de 830px)
- Listas sem marcadores (`list-style: none`)
- Botões customizados com CSS

---

### Exercício 03 — Layout Tetris

Um layout experimental que usa `div`s coloridas para simular o posicionamento de peças de Tetris na tela, cobrindo toda a viewport.

**Conceitos praticados:**
- Uso de unidades relativas `vw` e `vh`
- Flexbox com `flex-wrap`
- Composição de layout sem uso de imagens

---
